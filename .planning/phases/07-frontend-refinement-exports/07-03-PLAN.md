       ```
       apiKeys = pgTable('api_keys', {
         id: uuid('id').primaryKey().defaultRandom(),
         firmId: uuid('firm_id').notNull().references(() => firms.id, { onDelete: 'cascade' }),
         name: text('name').notNull(),
         keyHash: text('key_hash').notNull().unique(),
         lastUsedAt: timestamp('last_used_at', { withTimezone: true }),
         ...timestamps  // createdAt, updatedAt
       })
       ```

       - `POST /api/api-keys`: Generate `sk_live_${crypto.randomBytes(32).toString('hex')}`, store SHA-256 hash in DB, return the raw key in response (shown once). Accept `{ name: string }` body.

       - `DELETE /api/api-keys/:id`: Soft-delete or hard-delete the key. Use adminDb with firmId WHERE clause (defense-in-depth pattern, consistent with other routes).
       - Hash function: `crypto.createHash('sha256').update(rawKey).digest('hex')`

    1. Install packages in apps/api: `pnpm add @hono/zod-openapi@0.18.4 @scalar/hono-api-reference`
       IMPORTANT: Pin @hono/zod-openapi to 0.18.4 — v1.x requires zod v4 which is incompatible with the project's zod v3. Verify with `pnpm why zod` that only one zod version resolves.

    2. Create `apps/api/src/middleware/api-key-auth.ts`:
       - Extract `x-api-key` header
       - Validate it starts with `sk_live_`
       - Hash with SHA-256 and look up in api_keys table (using adminDb)
       - If found: set `c.set('firmId', apiKey.firmId)`, update lastUsedAt, call `next()`
       - If not found or missing: return `c.json({ error: 'Unauthorized' }, 401)`

    3. Create `apps/api/src/routes/v1/index.ts`:
       - Create `OpenAPIHono` instance
       - Register security scheme: `v1App.openAPIRegistry.registerComponent('securitySchemes', 'ApiKey', { type: 'apiKey', name: 'x-api-key', in: 'header' })`
       - Apply `apiKeyAuthMiddleware` to all routes: `v1App.use('/*', apiKeyAuthMiddleware)`
       - Mount sub-routes: `v1App.route('/deals', dealsV1Routes)`
       - Generate OpenAPI spec: `v1App.doc('/doc', { openapi: '3.0.0', info: { version: '1.0.0', title: 'J16Z Public API', description: 'Structured M&A deal data for programmatic access' } })`
       - Export `v1App`

    4. Create route files using `createRoute` from `@hono/zod-openapi`:

       `apps/api/src/routes/v1/deals.ts`:
       - `GET /` (maps to `/v1/deals`): List all deals for the firm. Query params: status (optional filter). Return array of deal objects.
       - `GET /:id` (maps to `/v1/deals/:id`): Single deal by ID. Return 404 if not found or wrong firm.
       - Define Zod schemas with `.openapi()` annotations for spec generation.
       - Use `adminDb` with explicit `firmId` WHERE clause (from `c.get('firmId')`).

       `apps/api/src/routes/v1/events.ts`:
       - `GET /` mounted at `/deals/:dealId/events`: List events for a deal. Verify deal belongs to firm first.

       `apps/api/src/routes/v1/clauses.ts`:
       - `GET /` mounted at `/deals/:dealId/clauses`: List clauses for a deal. Verify deal belongs to firm.

       `apps/api/src/routes/v1/spreads.ts`:
       - `GET /` mounted at `/deals/:dealId/spreads`: List market snapshots for a deal. Support `limit` query param (default 100). Verify deal belongs to firm.

       For sub-resource routes (events, clauses, spreads), mount them on the deals route in v1/index.ts as nested routes: `dealsV1Routes.route('/:dealId/events', eventsV1Routes)` etc.

    5. Mount v1 app and Scalar in `apps/api/src/index.ts`:
       - `import { v1App } from './routes/v1/index.js'`
       - `app.route('/v1', v1App)` — mount BEFORE the `/api` basePath routes
       - `import { Scalar } from '@scalar/hono-api-reference'` (or `apiReference` — check the actual export name)
       - `app.get('/docs', Scalar({ url: '/v1/doc' }))` — Scalar reads spec from /v1/doc endpoint
       - Note: The webhook route is already registered on root app pre-auth. Ensure /v1 mount doesn't conflict.

    6. Error handling: All v1 routes should return structured JSON errors: `{ error: string, status: number }`. Use Hono's `HTTPException` or manual `c.json()` for 400, 401, 404, 500.
