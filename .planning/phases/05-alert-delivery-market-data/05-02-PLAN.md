    - QuoteAdapter interface has getQuote(symbol: string): Promise&lt;QuoteResult | null&gt;
    - AlphaVantageAdapter fetches from GLOBAL_QUOTE endpoint with API key
    - AlphaVantageAdapter returns null on invalid symbol or rate limit (empty Global Quote object)
    - computeSpread for CASH deal: grossSpread = (offerPrice - currentPrice) / currentPrice * 100
    - computeSpread for STOCK deal: impliedConsideration = exchangeRatio * acquirerPrice
    - computeSpread annualized: (1 + grossSpread/100)^(365/daysToClose) - 1 (times 100 for percentage)
    - computeSpread returns { grossSpread, annualizedReturn, impliedConsideration }
    - isMarketOpen returns true Mon-Fri 9:30-16:00 ET, false otherwise

    1. Create `apps/api/src/market/types.ts`:
       - `QuoteResult` interface: { symbol: string, price: number, previousClose: number, volume: number, open: number, high: number, low: number, change: number, changePercent: string, timestamp: Date }
       - `SpreadComputation` interface: { grossSpread: number, annualizedReturn: number, impliedConsideration: number }
       - `QuoteAdapter` interface: { getQuote(symbol: string): Promise<QuoteResult | null> }

    2. Create `apps/api/src/market/quote-adapter.ts`:
       - `AlphaVantageAdapter` class implementing QuoteAdapter
       - Constructor takes apiKey string (from ALPHA_VANTAGE_API_KEY env)
       - `getQuote(symbol)`: fetch `https://www.alphavantage.co/query?function=GLOBAL_QUOTE&symbol=${symbol}&apikey=${apiKey}`
       - Parse response: `data['Global Quote']` — field names are `01. symbol`, `05. price`, `08. previous close`, etc.
       - Return null if quote is empty/missing (rate limit or invalid symbol)
       - Export `createQuoteAdapter()` factory that returns AlphaVantageAdapter with env var API key

    3. Create `apps/api/src/market/spread-calculator.ts`:
       - `computeSpread(currentPrice, offerPrice, exchangeRatio, acquirerPrice, daysToClose): SpreadComputation`
       - impliedConsideration: if exchangeRatio && acquirerPrice, use exchangeRatio * acquirerPrice; otherwise offerPrice
       - grossSpread = ((impliedConsideration - currentPrice) / currentPrice) * 100
       - annualizedReturn = (Math.pow(1 + grossSpread / 100, 365 / Math.max(daysToClose, 1)) - 1) * 100
       - `isMarketOpen(): boolean` — check if current time is Mon-Fri 9:30 AM - 4:00 PM ET. Use `Intl.DateTimeFormat('en-US', { timeZone: 'America/New_York' })` to get ET components. Return false on weekends.

    4. Create `apps/api/src/tests/spread-calculator.test.ts`:
       - Test CASH deal: $50 current, $55 offer, 30 days to close → grossSpread = 10%, annualized ~> 200%+
       - Test STOCK deal: exchangeRatio 0.5, acquirerPrice $100 → impliedConsideration = $50
       - Test edge case: daysToClose = 0 → uses Math.max(1) to avoid division by zero
       - Test negative spread (currentPrice > offerPrice)
       - Test isMarketOpen (mock Date for weekday in-hours, weekday after-hours, weekend)
